# cyberglobal.ai

Corporate website for **Cyber Global Technologies LLC** — Compliance & IT Risk Advisory.

Built as a static single-page site. Deployed on **Cloudflare Pages** at [cyberglobal.ai](https://cyberglobal.ai).

## Stack
- Static HTML / CSS / JS (no build step required)
- Google Fonts (Playfair Display, DM Sans, DM Mono)
- Calendly inline widget

## Deploy
Connected to Cloudflare Pages via GitHub. Any push to `main` triggers an automatic deployment.

**Build settings in Cloudflare Pages:**
- Framework preset: `None`
- Build command: *(leave empty)*
- Build output directory: `/` (root)
